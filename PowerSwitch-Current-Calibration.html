BlueBoat Current Sensor Calibration

This page is used to calibrate the BlueBoat's Power Switch current sensor offset based on the known expected current draw of the system. It checks current parameters, measures average current, and generates a new offset value. With just default hardware, we expect 0.5A quiescent current draw.
